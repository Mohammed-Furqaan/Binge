# 🎬 BINGE — Movie & Series Discovery Platform

BINGE is a modern movie and web series discovery platform that provides personalized recommendations based on user taste.  
It focuses on clean UI, smooth performance, and a real-world streaming experience.

## ✨ Features

- 🎯 Taste-Based Recommendations  
  Select genres, mood, content type, and language to get personalized suggestions  

- 🔥 Trending, Top Rated & Popular  
  Discover what’s currently popular and highly rated  

- 🎬 Cinematic Landing Page  
  Dynamic hero carousel with smooth transitions  

- 🔍 Search Functionality  
  Fast and responsive movie search  

- 🎲 Surprise Me  
  Get random recommendations instantly  

- ⚡ Performance Optimized  
  Lazy loading, optimized API usage, smooth UI  

- 📱 Fully Responsive  
  Works on mobile, tablet, and desktop  

---

## 🧠 Recommendation Logic

BINGE uses a weighted scoring system:

- Genre Match → High priority  
- Rating (>7) → Medium priority  
- Popularity → Additional weight  

This ensures relevant and high-quality recommendations.
